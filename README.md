# 📝 To-Do List App

A simple To-Do List application built using HTML, CSS, and JavaScript.

This project helps practice:

* DOM Manipulation
* Event Handling
* localStorage
* CRUD Operations

---

# 🚀 Features

* Add tasks
* Edit tasks
* Delete tasks
* Mark tasks as completed
* Filter tasks:

  * All
  * Active
  * Completed
* Save tasks in localStorage

---

# 💾 Local Storage

Tasks are automatically saved in the browser using:

localStorage


Tasks stay saved even after refreshing the page.
